# [자료구조] 1.배열 (Array)
- 같은 타입의 데이터를 여러개 나열한 선형 자료구조이다.
- 연속적인 메모리 공간에 순차적으로 데이터를 저장한다.
- 배열은 선언할 때 크기를 정하면, 그 크기로 고정이 된다.

```java
//new 키워드를 사용해서, 배열을 미리 선언하고, 데이터를 넣을 수도 있음
Integer[] data_list = new Integer[3];
data_lsit.add(10);
data_lsit.add(20);

```
